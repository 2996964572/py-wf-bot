

# 前言

随便写的一点功能，需要再添加
端口号可以在==main.py==自行修改
如果你需要成品的话可以直接在dist目录下下载

# 使用
如果你直接下在exe成品使用的话需要你先打开小栗子框架

导入**httpapi**插件

勾选打开上报地址

并添加上报地址：**http://127.0.0.1:886/** 并保存
然后q群回复**菜单**即可
